The definition of steady state temperature is the temperature which does not change with time. In a way, it is saying that the system is in equilibrium. Now let us revisit the governing equation. When the temperature has reached steady state implies that the time derivative of the temperature is zero. This implies that α*1000*A – ε *σ*A*T4 = 0. On solving this equation we get the desired result.
 
From the above relation the dependencies of the steady state temperature on various material properties and surface properties are clear. So as the ratio of α /ε increases the steady state temperature increases
 
Note this is not the temperature of the body always. The body starts at some initial temperature which can be totally independent of the steady state temperature. The variation of temperature from its initial value to its steady state value is the transient. The further the initial temperature from the steady state temperature, larger the time taken to reach steady state.
